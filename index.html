HTML Markup

For the html all you will need is a wrapper element and child elements that will act as the slides. The parent element could be a div or a ul tag. The child elements could be a div, however if you are using ul tags for the parent then you could substitute the child tags as li tags.

Options

				slide_parent : "#slider1", 		// Accepts the id of the parent element that contains the slides
				height : 300, 				// Specifies the height of inner sliders
				width : 800, 				// Specifies the width of inner sliders
				slides_shown : 1, 			// The amount of slides you want to show at one time. 
						             		// In this case only one slide will show at once on each rotation
				automatic : true		    	// Determines if slider will automatically rotate
				speed : 4000				// Accepts the amount of seconds that slider will rotate
				controls : true, 			// Determines if controller will be included
				control_prev : 'prev_arrow.png', 	// Specifies the file name of the previous button image
				control_next : 'next_arrow.png', 	// Specifies the file name of the next button image
				control_image_path : 'images/', 	// Contains the image path
